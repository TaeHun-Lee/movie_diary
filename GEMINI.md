## 프로젝트 개요

이 프로젝트는 사용자가 영화를 검색하고, 자신의 감상을 기록하는 영화 다이어리 애플리케이션의 프론트엔드, 백엔드 서버입니다. 프론트엔드는 Flutter 어플리케이션입니다. Flutter 프레임워크를 기반으로 구축되었으며, 사용자 인증, 게시글, 댓글 등 다양한 기능을 API로 사용합니다. 백엔드는 NestJS 프레임워크를 기반으로 구축되었으며, 사용자 인증, 게시글, 댓글 등 다양한 기능을 API로 제공합니다.

## 주요 화면
-   **로그인 화면:** 사용자 ID, PW를 입력 받아 로그인 하는 화면입니다.
-   **회원가입 화면:** 사용자 ID, PW, 닉네임을 입력 받아 회원가입 하는 화면입니다.
-   **홈 화면:** 내가 작성한 다이어리 목록, 내가 작성한 댓글 목록, 영화 검색하기 버튼이 있는 화면입니다.
-   **영화 검색 화면:** 영화 제목으로 검색하여 검색된 영화 목록을 확인할 수 있는 화면입니다.
-   **다이어리 작성/수정 화면:** 선택한 영화에 대해 다이어리를 작성 및 수정하는 화면입니다.
-   **다이어리 목록 화면:** 내가 작성했던 다이어리 목록을 전부 확인할 수 있는 화면입니다.
-   **다이어리 랭킹 화면:** 일정 기간 동안 가장 추천을 많이 받은 다이어리 목록들을 확인할 수 있는 화면입니다.
-   **회원정보 화면:** 내 정보 확인 및 수정을 할 수 있는 화면입니다.

## 주요 기능 및 모듈

-   **사용자 인증 (Auth):** JWT를 사용한 회원가입, 로그인, 프로필 조회 기능을 제공합니다.
-   **사용자 (Users):** 사용자 정보 조회, 수정, 삭제 기능을 담당합니다.
-   **영화 (Movies):** 외부 API(가상)를 통해 영화 정보를 검색하는 기능을 제공합니다. 영화에는 영화 제목, 감독, 간략한 내용, 영화 포스터 이미지 URL, 스틸컷 URL이 포함되어 있습니다.
-   **게시글 (Posts):** 사용자가 검색한 영화에 해당하는 영화 다이어리(게시글)를 생성, 조회, 수정, 삭제합니다.
-   **댓글 (Comments):** 특정 게시글에 대한 댓글을 생성, 조회, 수정, 삭제합니다.

## API 엔드포인트 명세

아래는 각 모듈별 주요 API 엔드포인트 목록입니다. 대부분의 API는 JWT 인증이 필요합니다 (`Bearer` 토큰).

---

### 인증 (Auth) - `/auth`

-   `POST /register`: 회원가입
-   `POST /login`: 로그인 (JWT 토큰 발급)
-   `GET /me`: 현재 로그인된 사용자 정보 조회 (인증 필요)

---

### 사용자 (Users) - `/users`

-   `POST /`: 새 사용자 생성 (회원가입 대용)
-   `GET /`: 모든 사용자 목록 조회
-   `GET /:id`: 특정 ID의 사용자 정보 조회 (인증 필요)
-   `GET /user_id/:user_id`: 특정 `user_id`의 사용자 정보 조회
-   `PATCH /:id`: 사용자 정보 수정
-   `DELETE /:id`: 사용자 정보 삭제

---

### 영화 (Movies) - `/movies`

-   `GET /search?title={title}&genre={genre}`: 영화 제목 및 장르로 검색 (인증 필요)
-   `GET /image?url={url}`: 이미지 프록시 (CORS 우회용)

---

### 게시글 (Posts) - `/posts`

-   `POST /`: 새 게시글 작성 (인증 필요)
-   `GET /`: 모든 게시글 목록 조회 (인증 필요)
-   `GET /my`: 내 게시글 목록 조회 (인증 필요)
-   `GET /popular`: 인기 게시글 목록 조회 (좋아요 순 상위 50개)
-   `GET /movie/:movieId/popular`: 특정 영화의 인기 게시글 목록 조회 (좋아요 순 상위 10개)
-   `GET /movie/doc/:docId/popular`: 특정 영화(DocId)의 인기 게시글 목록 조회 (좋아요 순 상위 10개)
-   `GET /:id`: 특정 ID의 게시글 상세 조회 (인증 필요)
-   `PATCH /:id`: 게시글 수정 (인증 및 작성자 확인 필요)
-   `DELETE /:id`: 게시글 삭제 (인증 및 작성자 확인 필요)

---

### 댓글 (Comments) - `/comments`

-   `POST /`: 새 댓글 작성 (인증 필요)
-   `GET /posts/:postId`: 특정 게시글의 모든 댓글 조회 (인증 필요)
-   `PATCH /:commentId`: 댓글 수정 (인증 및 작성자 확인 필요)
-   `DELETE /:commentId`: 댓글 삭제 (인증 및 작성자 확인 필요)
